Iniciando as funções JS

Inicie as funções JS do Locaweb Style quando necessário

Cada módulo JavaScript do Locaweb Style é iniciado individualmente caso nosso script detecte a presença do componente na página. Essa inicialização é feita automaticamente e você não precisa se preocupar com isso em ambientes normais.
Mas há momentos que você precisa reiniciar esse os módulos em ocasiões quando o DOM é modificado ou você está fazendo uma requisição AJAX e precisa que o componente seja executado novamente. Para tanto, você poderá inicar os módulos manualmente quando necessário.

Iniciando os módulos

Funciona assim: para iniciar um módulo use a seguinte chamada: locastyle.nome_do_modulo.init().
O exemplo abaixo inicia a nossa função de modal:

1
2
3
4
  (function() {
    // Inicia a função de Modal manualmente
    locastyle.modal.init();
  })();

Se você quiser iniciar todos os componentes do Locastyle, basta chamar a função: locastyle.init();.
Isso vai iniciar todos os módulos que estão sendo utilizados na página em que foi executada.
